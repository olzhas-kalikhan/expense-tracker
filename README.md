# Expense Tracker

## Startup:

Install dependencies `npm install`
Local startup `npm run dev`
Build `npm run build`
Preview build `npm run preview`

## Tools:

- TypeScript
- Vite
- React

## Libs:

- react-hook-form
- @tanstack/react-router
- @tanstack/react-table

## Styling:

- TailwindCSS
- shadcn-ui
